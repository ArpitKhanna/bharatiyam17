Code Matrix

Time to examine the quality of coding ability, technical knowledge and the perfection you have gained out of your engineering life. Here, we bring a full proof package for you to evaluate yourself among various fields. This event comprises of three rounds.
ROUND 1:- Code Teaser
This is the first round. Therefore, maximum number of participants will appear in this round. This round is kind of technical online quizzes to kick start your nut shells.
The quiz will be of 50 marks and divided into 4 sections containing 5 questions each. Section-A: - You will be given +1 for each correct answer and -0.5 for each wrong answer. Section-B: - You will be given +2 for each correct answer and -1 for each wrong answer. Section-C: - You will be given +3 for each correct answer and -1.5 for each wrong answer. Section-D: - You will be given +4 for each correct answer and -2 for each wrong answer. Rules:- 1. The time given will be 5 minutes for section-A and Section-B each, 7 minutes for Section-C and 8 minutes for Section-D. 2. Use of mobile phones and net services is strictly prohibited. ROUND 2:- Bugger As the name suggests, this round will test your coding ability in C environment. You will be given an erroneous coding. First of all, you will have to debug the code without changing the original logic. This code should be executable in any of the c compiler (turbo or gcc). Criteria of evaluation:- • Errors occurred while executing the code in compiler. • Number of omissions that will be occurred while handwriting your block of code. Rules:- • Time given will be 30 minutes. • The program executed in TC/GCC compiler must be a twin to the calligraphic code, if not you will be disqualified. • Use of pen drives or any external devices are strictly prohibited. • Use of mobile phones is not allowed. ROUND 3:- Code Panacea • Participants qualified from round-II will be selected for this round • Participants will be provided a specific problem statement and they are required to solve this problem by writing a code in given time. • Code with thorough logic and well defined features will be awarded. Rules: • Time given will be 30 minutes. • The program will be executed in TC/GCC compiler. • Use of pen drives or any external devices are strictly prohibited. • Use of mobile phones is not allowed.
